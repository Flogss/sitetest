Page d'accueil

Voici un paragraphe avec des hyperliens internes et externes.

HTML

Le HTML permet de décrire le contenu d'une page web à l'aide de balises.

  • Il peut exister plusieurs niveaux de titre.
  • Le texte est organisé en paragraphes.
  • On peut ajouter des hyperliens.
  • Et même des listes à puces !

CSS

Le CSS permet de modifier l'apparence du contenu.
On peut par exemple changer la couleur des titres.
